# AccessibilityTracker (working title)

A discovery tool for querying Google Maps Places API and filtering by wheelchair accessbile locations 

## The Problem
Currently, Google has a self-reporting feature where owners can upload details related to wheelchair accessibility: entrance, washroom, parking, seating. However, this is not searchable. You need to manually look up places and then see if they have a "♿" symbol next to it. This project aims to narrow down on an area and a type of place (restaurant, hospital) and then look through all places to list and rank the places that have been self-reported as WC accessible.
